# Portafolio SOC L1 – Nacho

Bienvenido a mi portafolio de alertas SOC L1.  
Aquí se documentan alertas detectadas, investigadas y escaladas, incluyendo acciones realizadas, decisiones y patrones identificados.  

## Organización
- 01-Alertas/: todas las alertas documentadas, organizadas por tipo.  
- 02-Templates/: plantillas para crear nuevas alertas.  
- imagenes/: evidencias gráficas o logs.  

Cada alerta tiene tags para filtrar por tipo (#credential_stuffing, #phishing, #dns_anomaly, etc.) y enlaces entre alertas relacionadas.
